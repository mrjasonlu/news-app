# News Co. App

Author: Jason Lu

## Enviroment Requirements:

- Node => 16
- NPM package manager

## Getting Started

### Install NPM packages

```bash
npm install
```

### start local environment

```bash
npm start
```

Browser should serve the app at `http://localhost:3000/`

## Running tests

```bash
npm test
```

## Production Build

```bash
npm run build
```
